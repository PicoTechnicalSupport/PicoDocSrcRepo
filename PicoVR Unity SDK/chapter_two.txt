2 	Requirements
=============================================

2.1 	VR Devices
-----------------------------

The SDK works with the following VR devices from Pico:

	* **VR All-in-one Headset**: Pico Goblin, Pico Neo, Pico Neo 2, Pico Goblin2, Pico Goblin2 4K



2.2 	Unity
---------------------------------------------------

The SDK is compatible with the following versions of Unity:

	* **Unity**: Unity 2017.4 to 2019.1

2.3	Android Development Environments
---------------------------------------------------

The SDK requires the following dependencies:

	* **JDK**: jdk1.8.0 and above
	* **Android SDK**: API Level 24 and above

2.4	Android Runtime
---------------------------------------------------
The following requirements must be satisfied if you develop Android mobile applications for Pico 1/1s or Pico U:

	* **Android Version**: Android 7.0 and above
	* **Required Sensors**: Gyroscope and accelerometer
