6 	API Reference
======================================

6.1 	General API
-------------------------------------

The SDK supports the generic VR functions marked with a red dot in Figure 6.1:

.. image:: _static/6.1.png

Figure 6.1 General API functions.

For more information, see the official `UE4 document <https://docs.unrealengine.com/en-us/>`_. 

**Note**: The **Reset Orientation and Position** node only supports resetting the positive direction and ignores the **Yaw** parameter, which is invalid.

6.2	Pico API
-------------------------------------

For system functions such as volume and brightness, the SDK provides the corresponding APIs in the form of blueprint nodes. Right-click in the event graph to enter the Pico Mobile sub-item and view the APIs:

.. image:: _static/6.2.png

Figure 6.2 System Interfaces.



6.2.1	Pico Get Current Orientation
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Returns the current orientation of the headset.

**Input**

None

**Output**

None

**Returns**

The current orientation of the headset.

6.2.2	Pico Get Current Position
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Returns the current orientation of the headset.

**Input**

None

**Output**

None

**Returns**

The current orientation of the headset.

6.2.3	Pico Get Sensor Count
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Returns the number of sensors in use.

**Input**

None

**Output**

The count of the sensors in use.

**Returns**

``True`` if the function succeeded or ``False`` if the function failed.

6.2.4	Pico Get Fov
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

**Input**

None

**Output**

None

**Returns**

The FOV.
 
6.2.5	Pico Get Current Brightness
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Returns the current system brightness level.

**Input**

None

**Output**

None

**Returns**

An integer indicating the current system brightness level, in the range of 0 through 255.

6.2.6	Pico Set Brightness
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Sets the system brightness level.

**Input**

The brightness level to set, in the range of 0 through 255.

**Output**

None

**Returns**

``True`` if the function succeeded or ``False`` if the function failed.

6.2.13	Pico Get Device Model
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Returns the device model type.

**Input**

None

**Output**

A string containing the device name and model string, separated with “---”. This will be set to one of the following values:

* ``Pico Neo --- Pico Neo``
* ``Pico Goblin --- Pico Goblin``
* ``Pico Goblin 2 --- Pico G2``
* ``Pico G2 4K --- Pico G2 4K``

**Returns**

Nothing

6.2.14	Get PSensor Status
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Gets the current PSensor Status.

**Input**

None

**Output**

Returns the following codes:

* ``-1``: Failed to get (uncheck "Enable PSensor"?).
* ``0``: Close to the device.
* ``1``: Away from the device.
