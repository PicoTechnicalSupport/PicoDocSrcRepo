2 	Instructions for SDK configuration
===========================================

2.1 	Supporting device 
-------------------------------


Pico Goblin，Pico Neo，Pico G2，Pico G2 4K 

2.2 	Development environment requirements
-------------------------------------------------

==============================================    ==================================================================
Software name                                      Version information
==============================================    ==================================================================
Unreal Engine                                     4.20.3、4.21.2 (recommended for Pico G2-4K), 4.22.3
Visual Studio                                     2017
Codeworks for Android                             The latest version is recommended. See https://docs.unrealengine.com/en-US/Platforms/Mobile/Android/InstallingAndroidCodeWorksAndroid for information on installing Codeworks.
==============================================    ==================================================================

The SDK has no special requirements for the ROM version of the hardware, but the latest version should be used when possible.

Be sure to check "Game development with C++" when installing Visual Studio 2017:
